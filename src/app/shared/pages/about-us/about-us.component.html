About this application

This application is a task management system built with Angular that allows users to view and organize their tasks efficiently. It provides separate sections for pending and completed tasks, along with detailed information about each task and the assigned user.

Features:

  • Pending Tasks: View a list of all tasks that are not yet completed, along with their details.
  • Completed Tasks: Browse tasks that have been finished.
  • Task Details: Click on any task to view more information, including its status and the assigned user.
  • User Information: See details about the user responsible for each task, including their name, email, phone number, and company.
  • Responsive Design: The application is optimized for different screen sizes, ensuring a smooth experience on both desktop and mobile devices.

Technology Stack:

  • Angular: Framework used for building the SPA with modular components.
  • TypeScript: Ensures type safety and better code maintainability.
  • Bootstrap: Provides a responsive and modern UI.
  • JSONPlaceholder API: The application fetches real-time task and user data from JSONPlaceholder.

How to Use:

Use the navigation menu to switch between pending and completed tasks. Click on any task to see more details, including the user responsible for it.
